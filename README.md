**Gestion du Personnel - WorkSphere**
##Description##

Ce projet est une application web interactive de gestion du personnel.
Elle permet d’ajouter, déplacer et supprimer des employés sur un plan d’étage en temps réel, tout en respectant les règles liées aux rôles et aux zones autorisées.
L’interface est moderne, intuitive et responsive, développée avec HTML, CSS et JavaScript.

##Objectifs##

Ajouter, déplacer et supprimer des employés directement depuis l’interface.

Assurer le respect des règles métier (ex. : seuls certains rôles peuvent accéder à certaines zones).

Offrir une expérience utilisateur fluide et responsive sur ordinateur, tablette et mobile.

Centraliser la gestion des données du personnel et leur visualisation spatiale.

##Fonctionnalités principales##

Liste des employés non assignés (“Unassigned Staff”).

Modale pour ajouter un nouvel employé (nom, rôle, photo, email, téléphone, expériences).

Plan d’étage avec 6 zones : Salle de conférence, Réception, Salle des serveurs, Salle de sécurité, Salle du personnel, Salle d’archives.

Attribution des employés aux zones selon les règles métier.

Suppression d’un employé d’une zone pour le renvoyer dans la liste non assignée.

Profil détaillé pour chaque employé (photo, informations, expériences, localisation).

Interface responsive avec animations CSS fluides.

Validation HTML et CSS via W3C Validator.

Déploiement sur GitHub Pages ou Vercel.

   ##Technologies utilisées

HTML5

CSS3 (Flexbox, Grid, Media Queries)

JavaScript (gestion dynamique et logique métier)

Git pour versionnement et gestion du projet
